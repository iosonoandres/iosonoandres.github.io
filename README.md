# Andres Camacho Portfolio

Portfolio personale sviluppato con React + Vite e pubblicato su GitHub Pages.

## Stack

- React 19
- Vite 6
- ESLint 9

## Comandi utili

```bash
npm install
npm run dev
npm run lint
npm run build
npm run preview
```

## Deploy

Il deploy pubblica la cartella `dist` su GitHub Pages.

```bash
npm run deploy
```
